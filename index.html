# Проэкт -> Git

Щас разберем порядок: когда ты создаешь с начала новый проект, а потом репозиторий, и тебе нужно:

-> связать НОВУЮ папку проэкта с новым репозиторием

Ниже - порядок команд в терминале по порядку

  • Правой кновкой на новой папке -> { Это все в проводнике Vs Code }
  • Открыть во встроенном терминале -> { Это все в проводнике Vs Code }
  • Дальше всё происходит уже в терминале

  • git init -> инициализация папки
  • git add . -> добавть в загрузку все файлы
  • git commit -m "Первый сайт или изменил стиль" -> описание твоего изменения в проэкте
  • git branch -M main -> меняешь ветку master на ветку main
  • git remote add origin https://github.com/ИМЯ-АККАУНТА-НА-ГИТЕ/ИМЯ-НОВОГО-РЕПОЗИТОРИЯ.git -> эту ссылку ты возьмешь после создания нового репозитория
  • git push -u origin main -> выгружаешь всё в новый реп.
